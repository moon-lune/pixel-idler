# pixel-idler
- 딱 idle 게임 정도의 방치 플레이 제공
- 너무 심심하지 않을 정도의 간단한 클릭 앤 드래그 컨텐츠 제공
- 도감 시스템 (장시간 플레이 할 거리)
- 오프라인 파밍 기능
- 키보드, 마우스 클릭에 반응하여 가속시키기.

- 초반: 골드, 성장 관련 답답함 해결
- 업그레이드 가격 밸런스
- 재료들간 획득, 소모 균일하게 밸런스
- 랜덤 인카운트 발생 > 수익

### 경험
**루틴**
- 입장 및 탈의실: 사복에서 헬스복으로 환복
- 워밍업: 스트레칭이나 런닝머신 가볍게 걷기
- 메인 운동 (예: 스쿼트 랙): 기구 탐색 -> 이동 -> \[15회 반복 -> 짧은 휴식] x 3세트
- 수분 및 단백질 보충: 자판기나 데스크로 이동해서 프로틴 드링크 결제 및 섭취 (유저 골드 획득 타이밍!)
- 샤워 및 탈의실
- 퇴장

실제로 픽셀 모션 시간을 정할때, 사람들이 추천하거나 정립되어있는 헬스 루틴 시간을 반영하면?
예를들어 스쿼트 모션 시간도 진짜 스쿼트 하는 시간대로 캐릭터가 움직이고, 움직이는 횟수가 10\~15개를 하고 1\~3분정도 쉰 뒤 다음 세트를 하고, 3세트가 끝나면 다른 기구로 이동하거나 운동을 마치고 샤워장으로 가거나 하는 등의 루트를 만드는것.

!!! 고쳐야 할 점
실제 헬스처럼 캐릭터가 1세트를 끝내고 진짜 현실 시간으로 1\~3분 동안 가만히 벤치에 앉아 쉰다면, 화면을 지켜보는 플레이어 입장에서는 게임이 멈춘 것 같거나 너무 지루하게 느껴질 수 있다.
'게임 속의 1분 = 현실의 1초'로 설정.
그러면 캐릭터가 10\~15개 동작을 리듬감 있게 수행하고, 현실 시간으로 1\~3초 정도 숨을 고르는 모습을 보여준 뒤 바로 다음 세트로 넘어감. 

더 재밌는 부분 -> 기구 부족 현상

예를 들면, 회원 20명 스쿼트랙 1개면 NPC들이 줄을 섬.
실제 헬스장처럼 "아 씨 또 스쿼트랙 없네" 발생.
그래서 플레이어가 스쿼트랙 추가 설치 > 수익 증가 경험.

# About this game
### Behind the scenes
"사실 저는 운동을 정말 싫어했습니다.
그런데 헬스를 사랑하는 사람을 만나면서 처음으로 헬스를 시작하게 됐고,
인연이 끝난 뒤에도 운동만큼은 제 삶에 남았습니다.
그래서 저는 헬스장을 운영하는 게임을 만들기로 했습니다."

'누군가를 좋아해서 시작(계기)한 일이 결국 내 인생의 일부(변화)가 되는 경험'

그 사람의 꿈중 하나가 자신의 헬스장을 운영하는 것이었다.
"나를 헬스의 세계로 이끌어준, 그리고 이 게임의 영감이 되어준 분에게 바칩니다."

'누군가가 선물했던 물건' 같은 작은 아이템이 헬스장 구석에 놓여있고, 그걸 클릭하면 잠시 스탯이 오르거나 특별한 메시지가 뜨는 식의 '이스터 에그'.
누군가가 좋아했던 운동 기구나 보충제 설명에 "어떤 사람이 헬스를 시작할 때 가장 좋아했던 기구" 같은 짧은 코멘트.
